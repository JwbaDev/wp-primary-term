A brief description of the plugin.

== Description ==

A WordPress plugin that enables you to easily designate any category, tag, or custom taxonomy term assigned to any post, page, or custom post type as the primary term.

== Installation ==

= Manual Installation =

1. Upload the entire `/primary-term` directory to the `/wp-content/plugins/` directory.
2. Activate Primary Term through the 'Plugins' menu in WordPress.
